Micronova Tasmota Driver

Berry script integration for Micronova-based pellet stoves on Tasmota.

Main files
  • Micronova.be (stable script)
  • MicronovaLatest.be (latest script)
  • EVENT_CODES.md (event/error code reference)

Quick install

  1. Open Tasmota Web UI on your device.
  2. Go to Console and enable Berry scripting if needed.
  3. Upload/copy Micronova.be into your script storage.
  4. Restart the device and verify JSON publish has a Stove object.

Available commands

StovePower ON|OFF
TargetWaterTemp <value>
TargetAmbientTemp <value>
TargetPower <1..5>
StoveWriteEEPROM <addr> <value>
StoveWriteRAM <addr> <value>
StoveReadEEPROM <addr>
StoveReadRAM <addr>

Telemetry

Published JSON contains a Stove object (live/target values) and periodic StoveEEPROM snapshots.
